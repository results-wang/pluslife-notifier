results.wang

results.wang sends you email notifications of PlusLife test results. It also allows you to see a similar graphs to the one virus.sucks shows, but using a link so you can see it from another computer or phone.

How to use it

You run the test in the virus.sucks unofficial analyzer. virus.sucks can forward its data to results.wang, which will email you on test completion.

While the test runs, and the computer/phone running the test must stay on, connected to the PlusLife device, and connected to the internet, until the test completes.
